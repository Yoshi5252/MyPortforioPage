# ポートフォリオサイト 仮組み作成プロンプト

## タスク概要
黒ベースのシンプルなポートフォリオサイトの大枠（仮組み）を作成してください。
後から自分で編集しやすいよう、コメントを丁寧につけてください。

---

## 技術要件

- **フレームワーク**: React + Vite
- **スタイリング**: Tailwind CSS（またはプレーンCSS）
- **ルーティング**: React Router v6
- **デプロイ**: GitHub Pages 対応（`vite.config.ts` の `base` 設定を含める）
- **言語**: TypeScript 推奨（難しければ JavaScript でも可）

---

## ページ構成

### 1. Home (`/`)
- キャッチコピー（仮テキストでOK）
- 自己紹介の一文
- 各ページへのナビゲーションリンク

### 2. Profile (`/profile`)
- 名前・所属（仮テキスト）
- スキルセット（箇条書き）
- SNS・GitHubリンク（仮URL）

### 3. Works - 写真 (`/works/photo`)
- 写真作品のグリッドギャラリー
- 画像はダミー（プレースホルダー画像 or 色付きdivで代替）

### 4. Works - エンジニアリング (`/works/engineering`)
- プロジェクトカード一覧
- カード内容：タイトル、概要、使用技術タグ、GitHubリンク（仮）

---

## デザイン要件

- **カラーテーマ**: 黒ベース（背景 `#0a0a0a` 〜 `#111111`）
- **アクセントカラー**: CSS変数 `--accent` で定義（仮は白 or グレー）
- **フォント**: Google Fonts から1〜2種（モダン系、日本語対応なら尚可）
- **レスポンシブ**: モバイル対応（最低限）

---

## 共通コンポーネント

- `Header.tsx` — ナビゲーションバー（Home / Profile / Works）
- `Footer.tsx` — シンプルなフッター
- `WorksNav.tsx` — Works内のタブ切り替え（写真 / エンジニアリング）

---

## ファイル構成（推奨）

```
src/
  components/
    Header.tsx       # ナビゲーション
    Footer.tsx       # フッター
    WorksNav.tsx     # Works内タブ
  pages/
    Home.tsx         # トップページ
    Profile.tsx      # プロフィール
    works/
      Photo.tsx      # 写真ギャラリー
      Engineering.tsx # エンジニアリング作品
  styles/
    global.css       # CSS変数・リセット・共通スタイル
  App.tsx            # ルーティング設定
  main.tsx           # エントリーポイント
```

---

## コメントに関する指示

以下の箇所には必ずコメントを入れてください：

1. **CSS変数の定義箇所** — 「ここでテーマカラーを変更する」など
2. **各コンポーネントの先頭** — 役割の説明
3. **ダミーデータの箇所** — 「ここに実際の〇〇を入れる」
4. **ルーティング設定** — 「ページを追加する場合はここに追記」
5. **GitHub Pages 用設定** — `vite.config.ts` の `base` の説明

---

## 注意事項

- 画像は外部URLや実ファイルは不要。`<div>` や `<span>` でプレースホルダーを代替
- 実装はあくまで「仮組み」でOK。デザインの完成度より構造の明快さを優先
- `npm run dev` で即起動できる状態にすること
- `npm run build` → `dist/` が GitHub Pages にそのままデプロイできること
